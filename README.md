# AI Resume Analyzer - RAG System

An intelligent resume analysis system using Retrieval-Augmented Generation (RAG) that helps job seekers optimize their resumes against job descriptions.

## Features

- 📄 **PDF Resume Upload** - Upload resumes in PDF format
- 🔍 **Semantic Search** - Uses vector embeddings for intelligent matching
- 📊 **Match Score Analysis** - Get percentage match with job descriptions
- 💡 **Improvement Suggestions** - AI-powered recommendations
- 🎯 **Keyword Gap Analysis** - Identify missing keywords from job descriptions
- 📑 **Source Attribution** - See which resume sections matched
